# Changelog

All notable changes to `laravel-nhif` will be documented in this file.

## Release v2.2.0 - 2026-05-05

- Add support for Laravel 13 (testbench 11.*)
- Extend CI matrix to include Laravel 13.*

## Release v2.1.0 - 2026-05-05

- Add support for Laravel 11 and 12
- Bump minimum PHP to ^8.2
- Update dev dependencies: orchestra/testbench, pest, collision, larastan (renamed from nunomaduro/larastan), spatie/laravel-package-tools, phpstan rules
- Expand CI matrix to PHP 8.2-8.4 x Laravel 10/11/12
- Fix service provider FQCN casing in composer.json (broke Linux PSR-4 autoload)

## Release v2.0.0 - 2023-03-21

- Add support for Laravel 10
- Drop support for Laravel 9 and earlier
- Bump minimum PHP to ^8.1

## Release v1.0.1 - 2022-07-28

- update documentation

## Release v1.0.0 - 2022-07-28

- Initial Release
